# 校园数字孪生三维交互系统

> 本项目是 **浙江树人学院** 期中作业

## 项目简介

校园数字孪生三维交互系统是一个基于 WebGL 和 Three.js 技术构建的沉浸式校园三维可视化平台。系统可在现代浏览器中直接运行，无需安装任何插件，为用户提供沉浸式的校园三维浏览体验。

### 技术栈

| 技术 | 说明 |
|------|------|
| Three.js R160 | 核心 3D 渲染引擎 |
| OrbitControls | 相机交互控制 |
| ES6 Module + ImportMap | 模块化管理 |
| CSS3 | 界面样式 |

## 功能介绍

### 1. 三维场景渲染

系统构建了完整的校园三维场景，包含：

- **建筑类**（6栋）：教学楼、图书馆、食堂、宿舍、体育馆、钟楼
- **道路类**（5条）：主干道、环校道路、连接道路
- **植被类**（32+处）：银杏树、灌木丛
- **辅助设施**：路灯、长椅、巴士站、雕塑

### 2. 视角控制

- **鼠标左键拖拽**：旋转视角
- **鼠标滚轮**：缩放场景
- **鼠标右键拖拽**：平移视角

### 3. 视角切换

提供两种预设视角一键切换：

| 视角 | 说明 |
|------|------|
| 透视视角 | 第三人称视角，适合观察建筑细节 |
| 鸟瞰视角 | 正交俯视视角，适合查看整体布局 |

### 4. 图层控制

可独立控制三类对象的显示/隐藏：

- 建筑组
- 道路组
- 植被组

### 5. 信息展示

点击任意建筑或设施，右侧弹出信息面板显示：

- 对象图标和名称
- 对象类型
- 详细描述
- 统计数据（楼层、面积、建造年份等）

### 6. 动画效果

校园巴士沿预设路径循环行驶，途经校园主要建筑和道路。

## 运行方式

### 方法一：使用 Python HTTP 服务器

```bash
# 进入项目目录
cd campus-digital-twin

# 启动 HTTP 服务器
python -m http.server 8080

# 浏览器访问 http://localhost:8080
```

### 方法二：使用 Node.js serve

```bash
npx serve .
```

> **注意**：由于项目使用 ES6 Module 和 ImportMap，必须通过 HTTP 服务器访问，不能直接双击打开 HTML 文件。
